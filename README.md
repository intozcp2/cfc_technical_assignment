# cfc_technical_assignment
## Requirements
On Window:
  * Python ^3.6
  * pip ^22.3

On Mac:
  * Python3
  * pip3

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install packages.

On Window:
```bash
pip install -r requirements.txt
```
On Mac:
```bash
pip3 install -r requirements.txt
```
